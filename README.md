# Ejercicios básicos de javascript

Este repositorio recoje una batería de ejercicios básicos de javascript.
Los ejercicios recogen:

- exercici1-1 - Arrow Functions
- exercici1-2 - Ternary operator
- exercici1-3 - Callbacks
- exercici1-4 - Rest and spread operator
- exercici1-5 - Array transformation
- exercici1-6 - Array loops
- exercici1-7 - Promises & async/await

## Descarga del proyecto

```
git clone https://github.com/EbdiMar/ejercicios-basicos-javascript
```

## Modo de uso

Para poder testear el código incluido en los diferentes archivos, se puede usar el comando:

```
node archivo a testear
```

Hay que tener en cuenta que no todos los archivos se podrán testear de esta manera debido a qué, por cuestión del enunciado, no todos los ejercicios son funciones sino expresions y, por lo tanto, al ejecutar una expresión puede dar error por consola.

## Tecnologías

- JavaScript vanilla
